2) ### Why react was created ? ###
=> React.js was created to build fast and efficient user interfaces for web applications. It solves the problem of slow DOM updates and complex UI management by using components and a Virtual DOM, making applications easier to develop and maintain.

=>When JavaScript data (state) changes, the DOM (UI) also needs to update to show the latest data.
In normal JavaScript, developers had to manually update the DOM, which was complex and error-prone.

Example problem:
JS state: count = 5
UI should display: 5
If state changes to 6, you must manually update the DOM.
If you forget to update the DOM, state and UI become out of sync.

React solves this problem by automatically syncing the state with the UI using the Virtual DOM and re-rendering.
